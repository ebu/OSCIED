Source Diff Tool
----------------

Here you can compare any versions of any files under |VCS|_. This feature is linked to one of the most interesting and intensively used feature of Subversion_ always used before *committing* anything. Useful for any developer to recall itself what he does in order to write the message of the *commit*.
